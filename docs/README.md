## 推荐阅读顺序

1. 先读需求分析，确认范围、玩法规则和验收标准。
2. 再读设计文档，理解架构、状态机、页面组织和实现建议。
3. 再读 UI Flow，检查每个手表操作流程。
4. 最后打开 HTML 预览原型，对照 432 x 514 画面节奏。
